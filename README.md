SAFAIR AI Threat Knowledge Base

This tool has been implemented by Tecnalia (https://www.tecnalia.com/) within the context of the SAFAIR program (Secure and Reliable AI Systems for Citizen)in the SPARTA project (Strategic Programs for Advanced Research and Technology in Europe.

The SAFAIR AI Threat Knowledge Base may be instantiated as a MySQL database. In order to connect to the MySQL Server, you can use the MySQL command-line client or the graphical MySQL Workbench.  Instructions for the required installations can be found at the MySQL website.
